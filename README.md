
## 我还是很喜欢你，像风走了八千里，不问归期

在这个流行“快餐式爱情”的时代，总有一些深情显得格格不入。比如我对你的喜欢，它不声不响，却漫长得看不到尽头。

### 01. [执念如风，不知所起，一往而深](https://www.hongtaosp.cloud)

你离开后的日子里，我的生活似乎早就恢复了平静。我照常工作、 social、和朋友大笑，所有人都以为我早就把你放下了。

可只有我自己知道，你就像是一阵风，虽然早已吹了过去，却在我的世界里留下了经久不息的余音。

* 听到一首熟悉的歌，会突然失神；
* 路过那家我们常去的咖啡馆，目光还是会不自觉地往里探寻；
* 看到一个和你相似的背影，心跳还是会漏掉一拍。

**“我还是很喜欢你。”** 这句话我再也没有对你吹过，我把它藏在日落里，揉进晚风里。它变成了一个只有我自己知道的秘密。

---

### 02. [像风走了八千里，跨越山海，不知疲倦](https://www.hlzw.cloud)

很多人问过我，既然已经分开了，既然知道没有结果，为什么还要这么固执？

我不知道该怎么解释。感情如果能被理智精准控制，那大概也就称不上是爱了。

> 喜欢你这件事，就像是一场没有终点的徒步。风吹过平原，翻过高山，穿过荒漠，它走过了八千里路云和月。它也会累，也会在某个寒冷的夜里冻结，但只要黎明一到，它又会不由自主地向着你的方向流动。

这种喜欢，已经不再奢求占有，也不再纠结于“你还爱不爱我”。它变成了一种本能，一种我与这个世界产生温柔连接的方式。因为喜欢过你，我才变得比以前更温柔、更懂得体谅、更懂得如何去爱。

---

### 03. [不问归期，是我的决绝，也是我的温柔](https://www.mrdspc.cloud)

这世上最奢侈的，莫过于“甘愿”二字。

**“不问归期。”** 这四个字，是我能给这段感情最后的尊严与温柔。

* 我不问你何时回头；
* 我不求你等价回报；
* 我甚至不奢望在你的未来里，还能有我的名字。

这就好比风吹向远方，它只是在履行自己的宿命，至于那棵被风吹动的树，是否会记得风的样子，已经不重要了。我将这场浩大的喜欢全盘托出，感动了自己，也放过了你。

---

### 04. [谢谢你，曾做过我的方向](https://www.51baoliao.cloud )

风走了八千里，它见过了星辰，见过了大海，也见过了这世间最美的风景。虽然它最终没有停留在任何一个地方，但这一路的旅程，已经足够辽阔。

正如我喜欢你一场。

虽然我们没能走到最后，虽然故事的结局写满了遗憾，但我不后悔。谢谢你曾惊艳过我的时光，谢谢你曾做过我逆风前行时的方向。

**往后余生，这阵风或许会慢慢停息，但在它彻底安静下来之前，它路过的每一寸土地，都曾写满我对你的心意。**
